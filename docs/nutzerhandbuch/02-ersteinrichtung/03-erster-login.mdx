Um sich das erste Mal in Ihrem Berater-Account einzuloggen, klicken Sie bitte auf den Link <UIElement>Passwort vergessen?</UIElement>, denn Sie auf der Login-Seite unter dem Anmelde-Button finden.

![Loginseite](../../../static/img/images-onlineberatung/02_Ersteinrichtung/03_1_Screenshot.png)

Anschließend öffnet sich ein neues Fenster, in dem Sie dann Ihre E-Mail-Adresse eingeben müssen. Weitere Informationen finden Sie im folgenden Abschnitt <UIElement>Passwort zurücksetzen</UIElement>.
