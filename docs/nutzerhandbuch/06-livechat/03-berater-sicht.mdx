## 1:1 Chat aktivieren

Um einen 1:1-Chat zu starten, muss ein Ratsuchender eine Anfrage stellen. Voraussetzung, dass ein Ratsuchender eine Anfrage stellen kann, ist, dass mindestens ein Berater / eine Beraterin verfügbar ist.
Berater_innen haben zwei Möglichkeiten, sich für den 1:1-Chat verfügbar zu melden:

<ul>
  <li>
    Gehen Sie zum Menüpunkt <UIElement>Profil</UIElement> und klicken Sie auf
    den Tab <UIElement>Meine Aktivitäten</UIElement>. Wählen Sie anschließend
    "Meine Live-Chat-Verfügbarkeit" aus und aktivieren Sie die entsprechende
    Umschalttaste.
  </li>
</ul>

![1:1 Chat](../../../static/img/images-onlineberatung/06_LiveChat/01_Screenshot.png)

<ul>
  <li>
    Gehen Sie zum Menüpunkt <UIElement>Erstanfragen</UIElement> und wechseln Sie
    zum Tab <UIElement>Live-Chat-Anfragen</UIElement>. Dort finden Sie die
    Option, die Verfügbarkeit zu aktivieren, indem Sie auf den Button
    <UIElement>Verfügbarkeit aktivieren</UIElement> klicken.
  </li>
</ul>

![1:1 Chat](../../../static/img/images-onlineberatung/06_LiveChat/02_Screenshot.png)

Durch eine dieser beiden Methoden können Berater_in ihre 1:1-Chat-Verfügbarkeit aktivieren und können dadurch 1:1 Chat Anfragen empfangen. Sie können den 1:1-Chat aktivieren, indem Sie eine der oben beschriebenen Methoden verwenden.

## 1:1 Chat Anfrage annehmen

Wenn Sie Ihre 1:1-Chat-Verfügbarkeit aktiviert haben und ein Ratsuchender stellt eine 1:1-Chat-Anfrage, sehen Sie diese Anfragen unter dem Menüpunkt <UIElement>Erstanfragen</UIElement> und dem Tab <UIElement>Live-Chat-Anfragen</UIElement>. Sie sehen an dieser Stelle, alle vorliegenden 1:1-Chat-Anfragen.

![1:1 Chat](../../../static/img/images-onlineberatung/06_LiveChat/03_Screenshot.png)

Da die Ratsuchenden sich nicht registrieren müssen, tragen alle den Benutzernamen "Ratsuchende_r" und werden einfach durchnummeriert, beispielsweise "Ratsuchende_r 4".

Wenn Sie den Ratsuchenden anklicken, wird die Anfrage geöffnet und Sie können die Anfrage annehmen:

![1:1 Chat](../../../static/img/images-onlineberatung/06_LiveChat/04_Screenshot.png)

Wenn Sie auf den Button <UIElement>Chat starten</UIElement> klicken, erhalten Sie folgenden Hinweis:

![1:1 Chat](../../../static/img/images-onlineberatung/06_LiveChat/05_Screenshot.png)

Wenn Sie nun auf den Button <UIElement>Antworten</UIElement> klicken, werden Sie automatisch zum Menüpunkt <UIElement>Meine Nachrichten</UIElement> weitergeleitet, wo Sie nun die Beratungskommunikation mit dem Ratsuchenden durchführen.

## 1:1 Chat durchführen

<PrimaryNote>
  So lange Sie als Berater_in die Chatanfrage nicht angenommen haben, kann der
  Ratsuchende Ihnen keine Nachricht schreiben.
</PrimaryNote>

Nachdem Sie eine 1:1-Chat-Anfrage angenommen haben, befindet sich die Beratungskommunikation unter dem Menüpunkt <UIElement>Meine Beratungen</UIElement>.

![1:1 Chat](../../../static/img/images-onlineberatung/06_LiveChat/06_Screenshot.png)

Sie haben jetzt die Möglichkeit, direkt mit dem Ratsuchenden zu kommunizieren. Sie können alle Kommunikationsmöglichkeiten nutzen, die Ihnen auch bei einer "normalen" Beratungskommunikation zur Verfügung stehen. Beachten Sie jedoch, dass es bei einem 1:1-Chat keine Videoberatung geben kann, da es sich um eine anonyme Beratung handelt

<PrimaryNote>
    Anhand des Namens des Ratsuchenden und des grünen Elements mit dem Text "Aktiv" können Sie erkennen, ob es sich um einen 1:1-Chat oder eine Textnachricht handelt.
    Wenn ein 1:1-Chat läuft, wird dies durch den Hinweis "aktiv" angezeigt. Sobald der Chat beendet ist, entfällt dieser Hinweis.
</PrimaryNote>

## 1:1 Chat beenden

<PrimaryNote>
  Nachdem der 1:1-Chat beendet wurde, kann dieser nicht wieder aktiviert werden.
</PrimaryNote>

Der Chat kann sowohl von Ihnen als Berater_in als auch von dem Ratsuchenden beendet werden. Wenn Sie den Chat beenden möchten, müssen Sie auf den Button <UIElement>Chat beenden</UIElement> klicken.

![1:1 Chat](../../../static/img/images-onlineberatung/06_LiveChat/07_Screenshot.png)

Anschließend erfolgt eine Sicherheitsabfrage, um sicherzustellen, dass Sie den 1:1-Chat wirklich beenden wollen.

![1:1 Chat](../../../static/img/images-onlineberatung/06_LiveChat/08_Screenshot.png)

Wenn Sie auf <UIElement>Abbrechen</UIElement> klicken, bleibt der Chat bestehen. Klicken Sie auf <UIElement>Chat beenden</UIElement> wird der 1:1 Chat geschlossen. Im Nachrichtenverlauf erscheint dann der Eintrag "Der Chat wurde beendet".

![1:1 Chat](../../../static/img/images-onlineberatung/06_LiveChat/09_Screenshot.png)

Sobald der Chat beendet wurde, wird der Ratsuchende umgehend automatisch abgemeldet und kann sich nicht wieder im System anmelden, wodurch er / sie keinen Zugriff mehr auf die Beratungskommunikation hat.

Für Sie als Berater_in bleibt der Chat noch für 48 Stunden unter <UIElement>Meine Nachrichten</UIElement> einsehbar. Nach Ablauf der 48 Stunden wird der 1:1 Chat dann auch bei für Sie als Berater_in gelöscht.
