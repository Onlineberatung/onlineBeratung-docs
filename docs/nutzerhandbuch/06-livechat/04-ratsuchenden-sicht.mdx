## Anonyme live Beratung aktivieren

Die anonyme live Beratung ermöglicht es Ratsuchenden in Kontakt mit einer Beraterin / einem Berater zu treten, ohne hierfür einen Registrierungsprozess durchlaufen zu müssen. Somit gewährleistet die anonyme live Beratung maximale Anonymität für Sie als Ratsuchenden.

## Anonyme live Beratung starten

Um eine anonyme live Beratung zu starten, müssen Sie eine Anfrage stellen. Voraussetzung, dass ein Ratsuchender eine Anfrage stellen kann, ist, dass mindestens ein Berater / eine Beraterin ihre Verfügbarkeit für die anonyme live Beratung aktiviert hat.
Sie gelangen zur anonymen live Beratung über den Warteraum <Bold>(abc.onlineberatung.net/beratung/warteraum)</Bold>, wobei abc für den Namen der Subdomain der Online-Beratungsplattform steht.

Anschließend sollten Sie diesen Inhalt sehen:

![Anonyme live Beratung](../../../static/img/images-onlineberatung/06_LiveChat/10_Screenshot.png)

Um in den Warteraum der anonymen live Beratung zu gelangen, müssen Sie zunächst die Datenschutzerklärung und die Nutzungsbedingung der Online-Beratungsplattform akzeptieren, indem Sie auf <UIElement>Ich bin einverstanden</UIElement> klicken.

Wenn Sie die Datenschutzerklärung und die Nutzungsbedingung akzeptiert haben, wird geprüft, ob ein Berater / eine Beraterin für die anonyme live Beratung verfügbar ist.

Wenn kein Berater / keine Beraterin verfügbar ist, erscheint Folgendes:

![Anonyme live Beratung](../../../static/img/images-onlineberatung/06_LiveChat/11_Screenshot.png)

Wenn ein Berater / eine Beraterin verfügbar ist, erscheint Folgendes:

![Anonyme live Beratung](../../../static/img/images-onlineberatung/06_LiveChat/12_Screenshot.png)

Dieser Bildschirm wird so lange angezeigt, bis ein Berater / eine Beraterin Ihre Anfrage annimmt. Eine zeitliche Begrenzung gibt es hier nicht! Sie haben die Möglichkeit zu warten, bis ein Berater / eine Beraterin verfügbar ist. Falls Ihnen die Wartezeit nach einer gewissen Zeit zu lange ist, wird Ihnen die Möglichkeit geboten, sich auf der Online-Beratungsplattform zu registrieren.

Sobald ein Berater / eine Beraterin Ihr Anfrage akzeptiert, werden Sie zur Online-Beratungsplattform weitergeleitet:

![Anonyme live Beratung](../../../static/img/images-onlineberatung/06_LiveChat/13_Screenshot.png)

Nun können Sie mit Ihrem Berater / Ihrer Beraterin kommunizieren, indem Sie Textnachrichten formulieren und senden.

![Anonyme live Beratung](../../../static/img/images-onlineberatung/06_LiveChat/14_Screenshot.png)

## Anonyme live Beratung beenden

<PrimaryNote>
  Nachdem die anonyme live Beratung beendet wurde, kann dieser nicht wieder aktiviert werden.
  Sie haben anschließend kein Zugriff mehr auf die Inhalte der
  Beratungskommunikation.
</PrimaryNote>

Der Chat kann sowohl von Ihnen als Ratsuchender als auch von Ihrem Berater / Ihrer Beraterin beendet werden. Wenn Sie den Chat beenden möchten, haben Sie zwei Möglichkeiten:

### Chat beenden

Um die anonyme live Beratung zu beenden, können Sie auf den Button <UIElement>Chat beenden</UIElement> klicken.

![Anonyme live Beratung](../../../static/img/images-onlineberatung/06_LiveChat/15_Screenshot.png)

Anschließend erfolgt eine Sicherheitsabfrage, um sicherzustellen, ob Sie die anonyme live Beratung wirklich beenden wollen.

![Anonyme live Beratung](../../../static/img/images-onlineberatung/06_LiveChat/16_Screenshot.png)

Wenn Sie auf <UIElement>Abbrechen</UIElement> klicken, bleibt der Chat bestehen. Klicken Sie auf <UIElement>Chat beenden</UIElement> wird die anonyme live Beratung beendet und Sie werden automatisch abgemeldet und zur Startseite weitergeleitet.

### Abmelden

Um die anonyme live Beratungzu beenden, können Sie auf <UIElement>Abmelden</UIElement> im Hauptmenü klicken. Anschließend wird die anonyme live Beratung beendet und Sie werden automatisch abgemeldet und zur Startseite weitergeleitet.

![Anonyme live Beratung](../../../static/img/images-onlineberatung/06_LiveChat/17_Screenshot.png)
