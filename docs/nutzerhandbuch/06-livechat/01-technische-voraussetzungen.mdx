<PrimaryNote>
  Bei der Funktion "Anonyme live Beratung" handelt es sich um eine optionale Zusatzfunktion.
  Um Zugriff auf diese Funktion zu haben, muss also Ihre
  Online-Beratungsplattform diese Zusatzfunktion gebucht haben.
</PrimaryNote>

Um die anonyme live Beratung bestmöglich durchführen zu können, benötigen Sie einen der folgenden Browser:

<ul>
  <li>Google Chrome</li>
  <li>Mozilla Firefox</li>
  <li>Microsoft Edge Chromium</li>
  <li>Apple Safari</li>
</ul>

<PrimaryNote>
  Es muss immer die aktuelle Browserversion des jeweiligen Anbieters installiert
  sein. Prüfen Sie daher Ihre Browser regelmäßig auf Updates.
</PrimaryNote>

Wenn der Ratsuchende oder Sie keine Videoberatung wünschen, besteht die Möglichkeit, eine Art "Telefon-Beratung" durchzuführen, bei der keine Videoübertragung erfolgt. Nur Sie als Berater_in können diese Aktion starten, indem Sie anstelle des Kamera-Symbols auf den Telefonhörer klicken.
