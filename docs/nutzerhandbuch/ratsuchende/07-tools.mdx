---
title: Tools
sidebar_position: 7
---

Wenn Tools auf der Plattform aktiviert sind und der/die Berater_in Tools für die/den Ratsuchenden aktiviert hat, dann sehen Ratsuchenden einen neuen Reiter “Tools”. Dort sehen sie eine Übersicht der Freigeschalteten Tools. Mit Klick auf das jeweilige Tool öffnet sich dieses in einem neuen Tab und kann von den Ratsuchenden genutzt werden.

Bei einigen Tools, können Berater_innen die Inhalte der Ratsuchenden erst dann einsehen, wenn Ratsuchende sich dazu entschließen die Inhalte mit ihnen zu teilen.

Diese Einstellung ist auf der Einstellungsseite des jeweiligen Tools verfügbar.
