Sie können sich in Ihrem Browser benachrichtigen lassen, sobald Sie eine neue Nachricht erhalten.  

Eine Mitteilung informiert sie über den Eingang einer neuen Nachricht.  

<p><img src={useBaseUrl('/img/images-onlineberatung/01_Berater_BrowserNotifications.png')} style={{ width: '400px' }}/></p>

<p><img src={useBaseUrl('/img/images-onlineberatung/01_Berater_BrowserNotifications2.png')} style={{ width: '400px' }}/></p>

Hierbei erhalten Sie nach dem Login von ihrem Browser die Aufforderung Benachrichtigungen zuzulassen oder zu blockieren. 

Je nach Browser können sie diese Einstellung auch nur für eine Sitzung, für immer oder einen bestimmten Zeitraum vormerken und Sie können die Entscheidung über die Browser Einstellungen jederzeit wieder ändern. 

Wenn Sie die Benachrichtigungen zulassen, erhalten sie auditive und visuelle Hinweise beim Erhalten neuer Erstanfragen und Nachrichten, auch wenn Sie gerade in einem anderen Browserfenster arbeiten. Diese erhalten Sie nur solange Sie auch in der Anwendung angemeldet sind.
